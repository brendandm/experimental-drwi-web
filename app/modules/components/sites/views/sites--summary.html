Practice Batch Import

Batch Uploads allow you to upload polygons for multiple practices simultaneously. After upload, each record will display as a separate practice tile.

FieldDoc accepts .zip files that archive ESRI .shp, .shx, .dbg and a .prj file. The project should be WGS1984. GeoJSON files must use the json or .geojson file extensions and follow the format described in this specification.

Batch Uploads do not support Z-Dimensions. If your file contains a Z-Dimensions, please remove it prior to uploading to avoid potential errors.

Batch importing is optimized for 500 practices or less. Refer to our help documentation for more guidance on batch imports and upload requirements.
